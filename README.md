## Setup Instructions
To get started with ZephyrX, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/1Bytetisme/ZephyrX.git
   cd ZephyrX
   ```
2. Install the necessary dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```
